# PasswordResolver
Password resolver app for MacOS

Add project to Xcode and hit Run

Tested on Macbook Pro 15 mid2015 2.2Ghz:

- 2 ~ 0s (variants: 16)
- 3 ~ 0s (variants: 64)
- 4 ~ 0s (variants: 256)
- 5 ~ 0s (variants: 1024)
- 6 ~ 0s (variants: 4096)
- 7 ~ 0s (variants: 16384)
- 8 ~ 0s (variants: 65536)
- 9 ~ 0s (variants: 262144)
- 10 ~ 0.2s (variants: 1048576)
- 11 ~ 1.1s (variants: 4194304)

- 13 ~ 17.9 (variants: 67108864)
